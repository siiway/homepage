# SiiWay 群规 (QQ)

本群实行 "**五个严禁**" "**五个适当**" "**三个鼓励**" "**五个其他**".

## 五个严禁

1. 严禁发送任何违反《中华人民共和国网络安全法》的内容
2. 严禁发送任何 未经证实 且 不标明其未证实状态 的信息，尤其是涉及政治、民族、宗教、灾情、疫情、股市
3. 严禁发送任何令人 严重不适 的内容，包括但不限于视频、图片、网络平台
4. 严禁任何形式的 阴阳怪气、人身攻击、人格侮辱、网络暴力、网络欺凌、煽动极端情绪、侵犯他人隐私、泄露个人信息、组织网络水军 等行为
5. 严禁大量 刷屏、灌水，包括但不限于 短时间内发送大量的 文字、图片，如一定要发，请合并转发 (对一条消息大量 / 反复贴表情回复也在此列)

## 五个适当

1. 可 适当 发送 自己网站的信息，但 次数不宜过多，内容不宜过长，以及 不违反上述规则
2. 可 适当 发送 广告，但 次数不宜过多，内容不宜过长，以及 不违反上述规则
3. 可 适当 与群友开玩笑，但 不宜过于频繁，底线不宜逾越，以及 不违反上述规则
4. 可 适当 与群友交换友链，但 不要过于频繁，以及 不违反上述规则
5. 可 适当 与群友玩梗，但 不要过于频繁，且 避免 玩 低质量烂梗及其图文视频 (如奶龙、苦命鸳鸯等)，以及滥用数字 "6" 以 敷衍回答 / 将严肃问题娱乐化，避免 不合时宜地玩梗，以及 不违反上述规则

## 三个鼓励

1. 我们 鼓励 分享任何 不违反上述规则 的内容、资源
2. 我们 鼓励 群友之间 互相学习、互相帮助
3. 我们 鼓励 积极发言、保持群内活力

## 五个其他

1. 关于 灰色内容 (如网络协议) 的讨论建议去 DC / TG
2. 如果你要使用的 Bot 支持 在私聊使用，且你要 大量使用 的情况下，请 转入私聊 以免影响聊天
3. 转发本群 比较敏感 的 聊天记录 之前，请首先 取得管理同意，且 后果自负
4. 搬史 (意义不大的转发内容) / 发情 请去专用子群，群号、Key、二维码 见其他公告
5. 如果需要在 大群 转发内容，在 内容加起来 过长 (超过普遍的 手机端一个屏幕) 的情况下，请使用 合并转发 避免影响聊天

# 补充

如出现 违反上述规则的行为，群主及管理员有权在 主观**合理**范围 内进行 任何形式的处置，包括但不限于警告、禁言、移出群聊、**在全网范围内封禁已知账号**

如果对上述内容 / 处置措施 有异议，请 私聊 **群主 / 管理**，禁止因对处置措施不满而占用公共资源

SiiWay 团队对 本群规 拥有 最终解释权，**同时适用于 本群规** *(SiiWay Core -> SiiWay Member -> Group Admin -> Group Member)*
