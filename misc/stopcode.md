# Stopcodes

## 公共

### `QQ_GROUP_BANNED`

QQ 群被封禁。

### `QQ_CHANNEL_BANNED`

QQ 频道（腾讯频道）被封禁。

### `DISCORD_SERVER_BANNED`

Discord 服务器被封禁。

### `TELEGRAM_GROUP_BANNED`

Telegram 群聊被封禁。

### `TELEGRAM_CHANNEL_BANNED`

Telegram 频道被封禁。

## 内部

### `INTERNAL_QQ_GROUP_BANNED`

内部 QQ 群被封禁。

### `INTERNAL_TELEGRAM_GROUP_BANNED`

内部 Telegram 群聊被封禁。

## 成员

### `CORE_MEMBER_QQ_BANNED`

Core 成员 QQ 账号被封禁。

### `CORE_MEMBER_DISCORD_BANNED`

Core 成员 Discord 账号被封禁。

### `CORE_MEMBER_TELEGRAM_BANNED`

Core 成员 Telegram 账号被封禁。
