# 关于我们

SiiWay Team 成立于 2020 年 10 月，是一个**纯技术兴趣驱动的开源团队**，活跃于 Discord、QQ、微信、飞书、Telegram 等平台。

团队以技术交流为核心，致力于构建高质量的开源项目，覆盖聊天机器人、身份认证、开发工具、文档平台等多个领域。

## 核心项目

- [**NextBridge**](https://github.com/siiway/NextBridge) — 多平台聊天桥接框架，支持 Discord / QQ / Telegram 等主流平台互连
- [**urlclash-converter**](https://github.com/siiway/urlclash-converter) — Clash ↔ Link 订阅转换工具，纯本地运行，支持多种协议
- [**prism**](https://github.com/siiway/prism) — 自托管 OAuth 2.0 / OpenID Connect 身份认证平台，基于 Cloudflare Workers 全球部署
- [**glint**](https://github.com/siiway/glint) — 团队协作待办事项应用，基于 Cloudflare Workers + D1
- [**vellum**](https://github.com/siiway/vellum) — 多仓库文档平台，兼容 VitePress Markdown，FluentUI 外壳
- [**icons**](https://github.com/siiway/icons) — SiiWay 官方图标库

更多项目请访问 [GitHub 组织主页](https://github.com/siiway)。

## 技术栈

TypeScript / Python / Go / Rust / Cloudflare Workers / Vue
