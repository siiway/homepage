# homepage

SiiWay 团队主页

https://siiway.org

## 常用命令

```shell
pnpm install # 安装依赖
pnpm run dev # 启动开发服务器
pnpm run build # 构建站点 (生成静态 HTML)
pnpm run preview # 预览构建后的站点
```
