# 贡献指南

本文档用于说明如何为 SiiWay 主页项目贡献内容与代码。

## 适用范围

- 文档内容修订（错别字、链接修复、内容补充）
- 页面结构与样式优化
- 构建与工具链配置改进

## 基本流程

1. Fork 仓库并创建分支。
2. 完成修改并通过本地检查。
3. 提交 Pull Request，说明修改目的与影响范围。
4. 等待维护者评审与合并。

## 提交建议

- 小步提交，避免一次 PR 混入多个不相关改动。
- 提交信息应清晰描述变更内容。
- 涉及规范变更时，请同步更新对应规范文档。

## 质量检查

提交前建议至少完成：

- 拼写与链接自检
- 本地预览页面确认显示正常
- 对应语言版本内容一致性检查（如适用）
