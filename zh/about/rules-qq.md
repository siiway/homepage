# SiiWay 群规 (QQ)

本群实行 "**六个严禁**" "**三个适当**" "**三个鼓励**" "**四个其他**".

## TL; DR (Too Long; Didn't Read)

**严禁 (红线，一碰就处理)**:
1. 违法内容
2. 未证实的高风险信息 (政、民族、宗教、灾情等)
3. 严重不适内容 (自残、血腥、暴力描述)
4. 反复 / 大量性暗示、发情、文爱 (对方未同意更不行)
5. 阴阳怪气、人身攻击、泄露隐私、拱火、学历 / 种族 / 年龄 / 性取向 等歧视
6. 大量刷屏 / 灌水 (请合并转发)

**适当允许**: 发自己网站 / 广告、开玩笑玩梗、宣泄情绪 (别过度、不适).

**鼓励**: 分享资源、互帮互学、活跃聊天.

**其他**: 灰色话题去DC / TG; Bot / 长内容私聊或转发群; 敏感记录转发需管理同意; 搬史去专用子群 (群公告有).

**处罚**: 轻度 -> 提醒 / 撤回; 反复轻度 -> 短禁; 中度 -> 长禁 / 踢; 严重 -> 踢 + 全网封. (中度+ 发 wss.moe/record)

有异议私聊管理申诉 (或 wss.moe/report).

最终解释权: SiiWay 团队 (Core > Member).

如有时间，请同时读完整版 (wss.moe/rules-qq) 和行为准则.

## 六个严禁

1. 严禁 发送 / 发布 / 传播 任何违反中华人民共和国法律法规、违反《网络安全法》及相关规定的内容
2. 严禁发送任何 未经证实 且 不标明其未证实状态 的信息，尤其是涉及政治、民族、宗教、灾情、疫情、股市等内容
3. 严禁发送任何 严重引人不适 的内容，包括但不限于视频、图片、文字描述、平台内容 (自残、血腥、暴力等)
4. 严禁 大量 / 反复 发送带有明显 性暗示 / 发情 性质的 文字、表情、图片，或在对方 未明确同意 的情况下对群友 发情 / 文爱
5. 严禁任何形式的 阴阳怪气、人身攻击、人格侮辱、网络暴力、网络欺凌、学历 / 种族 / 年龄 / 性取向 等歧视、煽动极端情绪、侵犯 / 泄漏 他人隐私信息、组织网络水军 等行为
6. 严禁大量 刷屏、灌水，包括但不限于 短时间内发送大量的 文字、图片，如一定要发，请合并转发 (对一条消息大量 / 反复贴表情回复也在此列)

## 三个适当

1. 可 适当 发送 自己的网站、项目、广告，但 次数不宜过多，内容不宜过长，以及 不违反上述规则
2. 可 适当 与群友 开玩笑 / 玩梗，但 不要过于频繁，且 避免 玩低质量烂梗 (如奶龙、苦命鸳鸯等) / 滥用简单回复敷衍回答 / 将严肃问题娱乐化 / 不合时宜地玩梗，底线不宜逾越，以及 不违反上述规则
3. 可 适当 在群中宣泄情绪，但避免可能 引人不适 的 极端情绪、行为 描述，以及 不违反上述规则

## 三个鼓励

1. 我们 鼓励 分享任何 不违反上述规则 的内容、资源
2. 我们 鼓励 群友之间 互相学习、互相帮助
3. 我们 鼓励 积极发言、保持群内活力

## 四个其他

1. 关于 灰色内容 (如网络协议) 的讨论建议去 DC / TG
2. 为避免 大量无意义内容刷屏，可 在私聊使用 Bot / 合并转发内容 / 在转发群转发内容，主群转发请控制在 合理长度
3. 转发本群 比较敏感 的 聊天记录 之前，请首先 取得管理同意，且 后果自负
4. 搬史 (意义不大的转发内容) 请去专用子群，群号、Key、二维码 见其他公告

## 补充

如出现 违反上述规则 的行为:

- **初犯 / 偶尔轻度违规** (如 少量擦边、少量无意义转发) -> 撤回消息 + 口头 / 私聊提醒 / 警告
- **轻度 但 反复影响群氛围** (如 反复发情擦边、刷屏、多次无差别乱转) -> 可直接 撤回消息 / 警告 / 短时禁言 (10 分钟 - 1 天) ，以快速 恢复正常聊天秩序
- **中度违规** (如 明显阴阳怪气、泄露他人隐私、传播敏感信息、多次累犯轻度) -> 禁言 1 - 7 天 或 移出群聊
- **严重违规** (如 违法、严重人身攻击、网络暴力、恶意拱火、泄露群内记录造成严重后果、组织水军等) -> 直接 移出群聊 + 禁止再次入群 + **在全网范围内封禁已知账号** (如 极严重违规，包括但不限于 其他子群 / 项目群、Discord、Telegram、GitHub 等)

一般处罚以 **教育与恢复秩序** 为主，管理员会 综合考虑 违规意图、影响范围、成员反馈 及 过往记录.

如果对上述内容 / 处置措施 有异议，请 私聊 **群主 / 管理** (或访问 https://wss.moe/report)，禁止因对处置措施不满而占用公共资源.

中度违规及以上处置将会公布在 https://wss.moe/record *(SiiWay GitHub 组织 Discussions 的 `处理记录` 类别)*.

SiiWay 团队对 本群规 拥有 最终解释权，**同时适用于 本群规** *(SiiWay Core -> SiiWay Member -> Group Admin -> Group Member)*.
